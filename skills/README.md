# ImgBin Skills

This directory is the repo-local AI documentation entrypoint for `repos/imgbin`.

Open `imgbin-cli/SKILL.md` first when you need structured guidance for the ImgBin CLI.

## What To Open

| Need | Open |
| --- | --- |
| Pick the right command and confirm flag guardrails | `imgbin-cli/SKILL.md` then `imgbin-cli/references/imgbin-cli-commands.md` |
| Configure providers, env vars, or timeouts | `imgbin-cli/references/imgbin-cli-config.md` |
| Understand managed library files and metadata side effects | `imgbin-cli/references/imgbin-cli-library-model.md` |
| Follow an end-to-end recipe | `imgbin-cli/references/imgbin-cli-workflows.md` |

## Scope

- These files document the current repository contract for `imgbin`.
- They are repo-local guidance; the first iteration is not part of the published npm package.
- Source code remains the final authority when behavior and docs disagree.
